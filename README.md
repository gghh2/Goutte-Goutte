GoutteÀGoutte vous accompagne dans la réhydratation de votre enfant lors d'une gastro-entérite, 
en suivant précisément le protocole Picolite recommandé par les pédiatres.

L'application calcule automatiquement les doses selon le poids de l'enfant, vous guide 
à travers les phases 1 et 2 du protocole, et vous alerte au bon moment pour chaque prise.

Fonctionnalités :
- Timer avec notifications persistantes
- Suivi des prises et des vomissements
- Graphiques de progression en temps réel
- Export des données pour le médecin
- Fonctionne hors ligne (PWA)
- Calculs automatiques selon le poids

Prenez soin de votre enfant, goutte à goutte.
